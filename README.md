# AST PROJECT
This is a node.js project for Asynchronous Server Technologies (ECE Paris course).

## Technology
<ul>
  <li>Node.js</li>
  <li>coffeeScript</li>
  <li>express & npm</li>
  <li>JADE</li>
  <li>Bootstrap</li>
</ul>

## Description
Web site with login and display graph to analyse Forex Market with an embedded database.
